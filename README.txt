1) This software uses MinGW GCC 6.1.0 compiler and Boost library.
   To install latest GCC compiler, download the distro from
   https://nuwen.net/mingw.html

2) The make file assumes the distro is installed at c:\MinGW14\MinGW
   If the distro in installed in any other place then update the make
   file accordingly before building the project.
